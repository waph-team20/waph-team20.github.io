Hackathon-1 Overview

The Cross-site Scripting (XSS) Hackathon is an educational exercise designed to explore and understand various levels of XSS vulnerabilities in web applications. XSS is a common security vulnerability that allows attackers to inject malicious scripts into web pages viewed by other users. The hackathon consists of two main tasks: Task 1 focuses on identifying and exploiting XSS vulnerabilities in a target web application, while Task 2 involves implementing defenses against XSS attacks to mitigate the risk of exploitation. The link to access hackathon1 is: https://github.com/sheelada/waph-sheelada /tree/main/Labs/hackathon1.
